## Prerequisites

- CUDA capable hardware
- python >= 3.7
- numpy >= 1.18.5
- pandas >= 1.1.0
- matplotlib >= 3.3.0
- pytorch >= 1.6.0+cu101
- tensorboard
- hpbandster
- ConfigSpace (**It must require Microsoft Visual C++ 14.0**)
- ray
- Pywavelet >= 1.1.1
